# 🌍 Explore India - Tourism Website

Explore India is a responsive tourism website developed using **HTML, CSS, and JavaScript**. It allows users to explore tourist destinations, browse hotels, book rooms, complete a demo payment, print booking receipts, submit reviews, and manage their profiles. This project is designed as a college mini-project and demonstrates frontend web development concepts.

## ✨ Features

- 🏠 Home Page
- 🔐 Login & Sign Up
- 🏨 Hotel Listings
- 📅 Booking System
- 💳 Demo Payment Page
- 🎫 Printable Booking Receipt
- ⭐ Reviews & Ratings
- 
- 🌙 Dark Mode
-
- ❓ FAQ Page
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
